## Description

Opencv with node js 

`docker build -f Dockerfile -t $DOCKER_USERNAME/ex-opencv-nodejs .`


## Running the container
docker pull $DOCKER_USERNAME/ex-opencv-nodejs:latest
docker run --rm -p 80:5000 $DOCKER_USERNAME/ex-opencv-nodejs:latest
